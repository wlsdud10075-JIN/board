# board 알림톡 템플릿 초안 (BizM 등록용)

> BizM 콘솔에 **아래 문구 그대로** 등록·승인받는다. 발송 msg 가 승인본과 글자까지 같아야 통과.
> 코드 정본 = `app/Support/AlimtalkTemplates.php`. 문구 수정 시 **여기 + 코드 + BizM 동시** 반영.
> 발신프로필 = **car-erp 와 공유**(같은 Bizm 계정, Jin 2026-07-12).

## board_region_inspection — 지역 검차 안내
- 수신자: 그 지역×날짜 배정 검차원(`InspectionAssignment`).
- 변수: `#{지역}` · `#{건수}` · `#{차량목록}`(가변, 개행 구분 차량번호 목록 — 길이 검수 주의).
- 유형: 기본형(선택안함). title 미사용.

```
[검차 안내] #{지역}

검차 대상 차량 목록입니다.

■ 지역: #{지역}
■ 대상 차량 #{건수}대
#{차량목록}

ssancar.com 에 검차 결과를 등록해 주세요.
```

예시 렌더(#{차량목록} = 개행 목록):
```
[검차 안내] 경기 화성

검차 대상 차량 목록입니다.

■ 지역: 경기 화성
■ 대상 차량 2대
12가3456
34나5678

ssancar.com 에 검차 결과를 등록해 주세요.
```
